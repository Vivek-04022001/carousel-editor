# Rebrand reference designs — DROP IMAGES HERE

Claude ko har design style ka reference chahiye taaki woh code me exactly match kar sake
(bilkul waise jaise purane `Content/` gray design ko match kiya tha).

## Kya daalna hai

Tumhare paas 3 design styles hain. Har style ke liye yahan images daalo:

- **Cover slide** ka image (zaroori)
- **1–2 body slides** ka image (zaroori — taaki text layout samajh aaye)
- **CTA / last slide** ka image (agar us style me alag CTA design hai)

## Naam kaise rakho (clear rehne ke liye)

Ek folder me sab daal do, in naamon se:

```
Content/rebrand/
  style1-cover.jpg
  style1-body.jpg
  style1-cta.jpg        (optional)
  style2-cover.jpg
  style2-body.jpg
  style3-cover.jpg
  style3-body.jpg
```

`style1/2/3` ki jagah apni category ka naam bhi rakh sakte ho
(e.g. `darkpsy-cover.jpg`, `lines-cover.jpg`).

## Ya phir

Agar images already kahin disk pe hain (Downloads, Canva export, etc.), to bas
us folder ka **full path** Claude ko bata do — move karne ki zaroorat nahi.
